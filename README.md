### Blog Posts

* **Useful Packages (alphabetically):**
  + **`{cowplot}`** ‚Äî plot arrangements, themes & annotations  ‚Üí  [wilkelab.org/cowplot](https://wilkelab.org/cowplot/articles/introduction.html)
  + **`{ggalt}`** ‚Äî alternative coords, geoms, stats & scales  ‚Üí  [github.com/hrbrmstr/ggalt](https://github.com/hrbrmstr/ggalt)
  + **`{gganimate}`** ‚Äî create animations  ‚Üí  [gganimate.com](https://gganimate.com/)
  + **`{ggforce}`** ‚Äî several interesting add-on features   ‚Üí  [ggforce.data-imaginist.com](https://ggforce.data-imaginist.com/)
  + **`{ggmaps}`** ‚Äî access to Google & Stamen maps  ‚Üí  [github.com/dkahle/ggmap](https://github.com/dkahle/ggmap)
  + **`{ggplotly}`** ‚Äî create interactive plots  ‚Üí  [plot.ly/ggplot2](https://plot.ly/ggplot2/)
  + **`{ggpubr}`** ‚Äî publication-ready plot in one line  ‚Üí  [github.com/kassambara/ggpubr](https://github.com/kassambara/ggpubr)
  + **`{ggraph}`** ‚Äî networks, graphs & trees  ‚Üí  [github.com/thomasp85/ggraph](https://github.com/thomasp85/ggraph)
  + **`{ggrepel}`** ‚Äî prevent overlapping text labels  ‚Üí  [github.com/slowkow/ggrepel](https://github.com/slowkow/ggrepel)
  + **`{ggstream}`** ‚Äî geoms for stream graphs  ‚Üí  [github.com/davidsjoberg/ggstream](https://github.com/davidsjoberg/ggstream)
  + **`{ggsankey}`** ‚Äî geoms for sankey diagrams  ‚Üí  [github.com/davidsjoberg/ggsankey](https://github.com/davidsjoberg/ggsankey)
  + **`{ggbump}`** ‚Äî geoms for bump charts  ‚Üí  [github.com/davidsjoberg/ggbump](https://github.com/davidsjoberg/ggbump)
  + **`{ggridges}`** ‚Äî geoms for ridgeline plots  ‚Üí  [github.com/clauswilke/ggridges](https://github.com/clauswilke/ggridges)
  + **`{ggtext}`** ‚Äî rich-text rendering  ‚Üí  [github.com/clauswilke/ggtext](https://github.com/clauswilke/ggtext)
  + **`{ggthemes}`** ‚Äî  additional themes, sclaes & geoms ‚Üí  [github.com/jrnold/ggthemes](https://github.com/jrnold/ggthemes)
  + **`{hrbrthemes}`** ‚Äî typography-centric themes  ‚Üí  [github.com/hrbrmstr/hrbrthemes](https://github.com/hrbrmstr/hrbrthemes)
  + **`{lemon}`** ‚Äî axis & legend add-ons  ‚Üí  [github.com/stefanedwards/lemon](https://github.com/stefanedwards/lemon)
  + **`{patchwork}`** ‚Äî combine ggplots  ‚Üí  [github.com/thomasp85/patchwork](https://github.com/thomasp85/patchwork)
  + **`{rayshader}`** ‚Äî hillshaded maps in 2D & 3D  ‚Üí  [github.com/tylermorganwall/rayshader](https://github.com/tylermorganwall/rayshader)
  + **`{systemfonts}`** ‚Äî use custom fonts  ‚Üí  [github.com/r-lib/systemfonts](https://github.com/r-lib/systemfonts)
